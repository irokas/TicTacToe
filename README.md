# Getting Started with Tic Tac Toe App

## Available Scripts

In the project directory, you can run:

### `yarn start`

Start express server, build react app and serve it!

Open [http://localhost:5000](http://localhost:5000) to view it in the browser.

### `yarn test`

Launches jest tests
